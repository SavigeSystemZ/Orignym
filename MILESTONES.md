# Milestones

This document records the completed and planned milestones for the Originym project.

## Completed Milestones (M0 - M7)

*   **M0: Project Foundation** - Documentation, system initialization, and repository setup.
*   **M1: Data Model & Auth** - Core PostgreSQL schema, Prisma integration, and NextAuth setup.
*   **M2: Validation Pipeline v1** - Staged verification system (exact, near, phonetic, semantic).
*   **M3: Suggestion Engine** - AI-powered alternative term generation based on verification context.
*   **M4: Publish & Registry** - Public-facing registry search and term viewing.
*   **M5: Moderation** - User reporting flows and administrative moderation queues.
*   **M6: Distribution** - Installation scripts and desktop application packaging.
*   **M7: Hardening & Production** - Real AI provider integration (Gemini/Claude) and Docker production readiness.

## Active Milestone

*   **M8: Full GUI Implementation & Design System**
    *   *Objective:* Build a cohesive, polished user interface covering all planned screens and flows (landing, onboarding, dashboard, draft editor, validation results, publish page, search, report/dispute, moderation, admin). Establish a reusable component library.

## Planned Milestones

*   **M9: Advanced Search & Evidence Infrastructure**
*   **M10: Advanced Suggestion Engine & Personalisation**
*   **M11: Collaboration, API & Enterprise Features**
*   **M12: Machine Learning, Legal Integration & Community**
