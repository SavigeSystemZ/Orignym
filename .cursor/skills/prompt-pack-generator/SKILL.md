# Prompt Pack Generator

## Inputs

- canonical docs
- milestone or phase goals
- required deliverables

## Steps

1. Reference the canonical docs by file name.
2. Write prompts for planning, implementation, validation, and handoff.
3. Require minimal diffs, no secrets, and explicit validation.
4. Keep prompts copy-paste ready.
