# Flutter Module

This is a minimal foundation, not a full production app.

## First steps

1. Run `flutter pub get`.
2. Configure the API host and verify package ids.
3. Add app icons and signing config.
4. Wire the UI to your actual runtime service or API.

## Flavors

- `dev` for local or QA endpoints
- `prod` for release endpoints

## Commands

- `flutter run`
- `flutter build apk --flavor prod`
- `flutter build appbundle --flavor prod`
